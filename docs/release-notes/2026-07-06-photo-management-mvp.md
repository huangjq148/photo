# Photo Management MVP Release Notes

## 交付范围

- 账号体系：注册、登录、退出、当前用户
- 空间体系：个人空间、空间列表、空间详情
- 照片体系：上传、列表、搜索、预览、下载、收藏
- 管理体系：删除到回收站、恢复、永久删除
- 批量能力：批量删除、批量恢复、批量永久删除
- 相册体系：创建相册、查看相册、相册内添加/移除照片
- 共享能力：公开分享链接、公开访问页
- 上传增强：多文件队列、取消、失败重试

## 验证结果

- `npm test` 通过
- `npm run build` 通过

## 备注

- 数据库迁移已同步到最新 schema
- 本地文件存储已接入原图、预览图、缩略图三份落盘
- 当前版本可作为 MVP 交付基线
